Sortable Coding Challenge - Alexander Mueller

Instructions for use:

https://github.com/alexandermueller/SortableCodingChallenge2015

1. Clone the repo at the above link, or type: 
"git clone git@github.com:alexandermueller/SortableCodingChallenge2015.git" in the terminal.
2. Place the “listings.txt” and “products.txt” files inside the "InputData" folder.
3. cd into the repo and type "./MatchListings.py" to run the program.
4. results.txt will be generated and put in the "ResultingData" folder.

Note: 

Missing files will be reported and the application will not run.
Some stats will be generated for your convenience and printed out to the terminal if successful.
